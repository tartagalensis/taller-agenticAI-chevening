# Taller de Agentic AI — Chevening 2025-26

Workshop corto (30-45 min) sobre cómo usar herramientas de **agentic AI** en investigación, trabajo de policy, y tareas profesionales reales.

## Qué vamos a hacer

1. **Charla intro** — qué es agentic AI, qué cuidados tener (~15-20 min).
2. **Demo en vivo** — el agente trabajando sobre un proyecto real de alguien en la sala (~10 min).
3. **Manos a la obra** — cada uno arranca su propio proyecto desde el template (~10 min).

## ⚠️ Antes del taller — hacé el setup (~15 min)

Llegá con esto **funcionando**, no lo vamos a hacer en vivo.

1. **Instalá Gemini CLI según tu sistema operativo:**
   - [macOS](./setup/setup-gemini-cli-mac.md)
   - [Windows](./setup/setup-gemini-cli-windows.md)
2. **Instalá [VSCode](./setup/setup-vscode.md)** y las extensiones recomendadas.
3. **Verificá que funciona:** abrí terminal, corré `gemini`, mandale "hola, qué podés hacer?". Si responde, listo.

Si te trabaste en algún paso, abrí un issue en este repo o avisame antes del taller. Mejor resolverlo sin presión.

## Durante el taller

Vas a usar [`project_template/`](./project_template/) como punto de arranque para tu propio proyecto. No hace falta tocar nada antes.
